## 1.js 引擎主要由两部分组成：

内存堆：这是内存分配发生的地方
调用栈：这是你的代码执行时的地方  
##2.调用栈##
JavaScript 是一门单线程的语言，这意味着它只有一个调用栈，因此，它同一时间只能做一件事。  
_调用栈是一种数据结构，它记录了我们在程序中的位置。如果我们运行到一个函数，它就会将其放置到栈顶。当从这个函数返回的时候，就会将这个函数从栈顶弹出，这就是调用栈做的事情。_

每一个进入调用栈的都称为调用帧。  
"堆栈溢出"，当你达到调用栈最大的大小的时候就会发生这种情况  
`function foo() { foo(); } foo();`
